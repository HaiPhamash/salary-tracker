# Salary Tracker

Multi-language salary and shift tracking app built with Capacitor for iOS and Android.

## What This Repo Contains

- `www/` — Main app source (HTML, CSS, JavaScript)
- `ios/` — Native iOS project
- `android/` — Native Android project
- `assets/` and `resources/` — App icons, splash assets, and source artwork
- `store/` — App Store / Play Store listing assets and legal copy
- `index.html`, `privacy-policy.html`, `support.html` — Public web pages hosted on GitHub Pages

## Features

- Multi-language interface
- Shift and salary tracking
- Monthly and period-based reports
- Export and sharing support
- Offline-first local data storage

## Development

Requirements:
- Node.js
- Xcode for iOS builds
- Android Studio for Android builds

Install dependencies:

```bash
npm install
```

Useful commands:

```bash
npm run sync
npm run sync:ios
npm run sync:android
npm run open:ios
npm run open:android
```
